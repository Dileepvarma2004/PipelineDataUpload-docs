# Dashboard Overview

The dashboard is your central hub for monitoring system activity and accessing key features.

## üìä Dashboard Layout

### Statistics Cards
The top section displays key metrics:

- **Total Pipelines**: Count of all pipeline records in the system
- **Active Today**: Number of activities performed today
- **Pending Updates**: Recent data uploads in the last 7 days
- **Alerts**: Error notifications from the last 3 days

### User Information Panel
- **Welcome Message**: Personalized greeting with your name
- **User Role**: Your current permission level (Admin, CreateModify, Standard User)
- **Last Login**: Timestamp of your previous session

### Navigation Sidebar
Quick access to all application features:
- Dashboard (current page)
- Create/Modify Data
- Bulk Upload
- Piping Data
- Data Approvals
- Manage Users (Admin only)
- Logout

### Calendar Widget
- **Monthly View**: Visual calendar with event indicators
- **Event Management**: Add, view, and delete scheduled events
- **Date Selection**: Click dates to view events for that day

### Notifications Panel
- **Recent Notifications**: Latest system and user notifications
- **Notification Types**: Upload confirmations, approval requests, system alerts
- **Action Items**: Notifications requiring your attention

## üìà Understanding Statistics

### Total Pipelines
- Represents the complete count of pipeline records
- Updated in real-time as new data is approved
- Includes all active pipeline segments

### Active Today
- Shows user activities for the current day
- Includes logins, uploads, data modifications
- Helps track daily system usage

### Pending Updates
- Recent data uploads requiring approval
- Time window: Last 7 days
- Indicates workflow backlog

### Alerts
- System errors and warnings
- Time window: Last 3 days
- Critical issues requiring attention

## üìÖ Calendar Features

### Event Types
- **Data Upload Events**: Track when data was uploaded
- **Approval Deadlines**: Important dates for pending approvals
- **Maintenance Schedules**: Planned system maintenance
- **Custom Events**: User-created reminders

### Managing Events
1. **Add Event**:
   - Click on a date in the calendar
   - Fill in event details (title, type, time, notes)
   - Save to create the event

2. **View Events**:
   - Click on dates with event indicators (dots)
   - View event list for selected date
   - See event details and timing

3. **Delete Events**:
   - Click the delete button next to any event
   - Confirm deletion when prompted

## üîß Troubleshooting Dashboard Issues

### Common Problems
- **Statistics not updating**: Refresh the page or check internet connection
- **Calendar not loading**: Clear browser cache or try different browser
- **Notifications not appearing**: Check notification settings
- **Sidebar not responding**: Try collapsing/expanding or refresh page

### Performance Issues
- **Slow loading**: Check internet speed or try different browser
- **Memory usage**: Close other browser tabs
- **Display issues**: Ensure browser is updated to latest version
